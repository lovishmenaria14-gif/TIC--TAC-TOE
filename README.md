# 🎮 Tic Tac Toe

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**. This project allows two players to play Tic Tac Toe in the browser with automatic winner detection and game reset functionality.

## 🎮 How to Play

1. The game starts with **Player O**.
2. Players take turns clicking on empty boxes.
3. The first player to align three symbols:
   - Horizontally
   - Vertically
   - Diagonally

   wins the game.
4. Click **Reset Game** to start a new match.
